หลักสูตรวิศวกรรมศาสตรบัณฑิตสาขาวิชาวิศวกรรมสารสนเทศและเครือข่าย

Bachelor of Engineering Program in Information and Network Engineering

มหาวิทยาลัย : มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าพระนครเหนือ วิทยาเขตปราจีน

คณะ : เทคโนโลยีและการจัดการอุตสาหกรรม

ระดับการศึกษา : ปริญญาตรี รหัสหลักสูตร : 25620151100293

หลักสูตรวิศวกรรมศาสตรบัณฑิต สาขาวิชาวิศวกรรมสารสนเทศและเครือข่าย (วศ.บ. สารสนเทศและเครือข่าย) ที่ มจพ. (มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าพระนครเหนือ) เป็นหลักสูตรที่มุ่งเน้นการผลิตบัณฑิตที่มีความรู้ความสามารถด้านเทคโนโลยีสารสนเทศ โดยเฉพาะอย่างยิ่งในด้านเครือข่ายคอมพิวเตอร์และระบบสารสนเทศ หลักสูตรนี้จะเน้นการเรียนรู้ทั้งภาคทฤษฎีและปฏิบัติ เพื่อให้บัณฑิตสามารถนำความรู้ไปประยุกต์ใช้ในการพัฒนาและบริหารจัดการระบบสารสนเทศและเครือข่ายได้อย่างมีประสิทธิภาพ
